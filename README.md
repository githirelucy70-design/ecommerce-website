# 🛒 E-commerce Website

## 📌 Description
This is a simple e-commerce web application developed using PHP and MySQL. The system allows users to browse products, add items to a cart, and provides an admin panel for managing products.

## 🚀 Features
- View available products
- Add products to cart
- Admin panel to add new products
- Dynamic data from MySQL database

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML
- CSS

## 💻 How to Run Locally
1. Install XAMPP
2. Start Apache and MySQL
3. Place project folder in `htdocs`
4. Import database in phpMyAdmin
5. Open `http://localhost/ecommerce`
